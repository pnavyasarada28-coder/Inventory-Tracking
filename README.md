📦 Inventory Tracking System

A full-stack Inventory Management Dashboard built using Django, designed to manage products efficiently with a clean UI and secure backend integration.

🚀 Features
📊 Dashboard with:
Total products
Low stock count
Category insights

📦 Product Management:
Add, Edit, Delete products (CRUD operations)
🔍 Search functionality
🏷️ Category-based filtering
⚠️ Status tracking (In Stock / Low / Out of Stock)
🌙 Dark & ☀️ Light theme (with persistence using localStorage)
🔐 Secure backend with CSRF protection
🔄 Dynamic frontend using JavaScript fetch API

🛠️ Tech Stack
Frontend
HTML
CSS (Bootstrap)
JavaScript
Backend
Django
Database
SQLite

⚙️ Key Functionalities
Built REST-style API endpoints using Django views
Implemented full CRUD operations with database persistence
Connected frontend to backend using asynchronous fetch requests
Handled CSRF protection for secure API communication
Designed modular UI with section-based rendering (Dashboard, Products, Settings)
Implemented theme switching with localStorage
